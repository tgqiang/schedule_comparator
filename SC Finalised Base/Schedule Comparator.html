SCHEDULE COMPARATOR

Welcome to Schedule Comparator


To start off for using this comparator, click "Sessions" in above menu to establish a session.
Not sure how to use the Schedule Comparator? Click here to learn more.


How do I use this comparator?


Step 1: Create/Join session

Go to "Sessions" in above menu to create a session for your group of friends or join an existing session created by one of your friends. This would then allow you to start using the Schedule Comparator.


Creating the session

In the "Sessions" page, click on the option for creating the session. Specify a desired session ID (which should then be told to only your group members) and the number of people allowed in the session and go ahead to create the session.


Joining the session

If you are not the one who created the session, you should instead join an existing session created by one of your friends, who should have informed you about the session ID over chat. Use that session ID to join the correct session and your group members can start filling in your schedules.


Step 2: Start using the comparator

There are two options offered in this comparator.


Manual Entry Comparison

For the manual entry comparison, you enter the dates that you are available for your group's events via the calendar date-picker interface in this website. Your friends or group members should also do the same and then the program would help you to find out the dates and times where all of you are available to meet up. For the calendar interface, simply click on the desired date once to select it. Click on the selected date again if you want to remove it from your selection.


NUSMods' Timetable Comparison

For the NUSMods' timetable comparison option, simply get everyone in your group to paste in the URL of your timetable in the input field and let this application do the rest.


Step 3: After you are done...

Either destroy the session or leave the session, then click on the option indicating that you wish to stop using this application.
